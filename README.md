# AI-Based Credit Scoring System

A full-stack AI-powered web application that predicts loan approval and generates a credit score using Machine Learning.

## Tech Stack

- React
- FastAPI
- Python
- Scikit-learn
- MySQL
- Tailwind CSS

## Features

- User Authentication
- Loan Application
- Credit Score Prediction
- Risk Analysis
- Admin Dashboard
- Prediction History
- Interactive Charts
